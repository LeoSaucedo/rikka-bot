# rikka-bot
A Discord bot that performs various functions.

### Commands
*Right now, this bot's prefix is* `;`*.*
- `;hello` - Says hi.
- `;hello (text)` - translates a given text with gizoogle.

### Requirements
- bs4
- requests
- lxml
