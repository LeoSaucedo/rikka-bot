# rikka-bot
A Discord bot that performs various functions.

### Commands
*Right now, this bot's prefix is* `;`*.*
- `;hello` - Says hi.
- `;gizoogle (text)` - translates a given text with gizoogle.
- `;hugme` - hugs you.
- `;hug @user` - hugs the specified user.

### Requirements
- bs4
- requests
- lxml
